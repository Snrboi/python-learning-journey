## 🤔 Challenges

At first it felt strange that nothing happens when an `if` condition is `False`, but I learned that Python simply skips the indented block.

---

## 💡 Key Takeaway

An `if` statement evaluates a condition. If the condition is `True`, Python executes the indented block of code. If it is `False`, Python skips it.
