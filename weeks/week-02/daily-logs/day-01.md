## 🧩 Bug Detective

Today I debugged another Python program.

I learned that not every issue in code is a syntax error—sometimes the code runs but produces different results depending on the operator used.

## 💡 Biggest Lesson

The same numbers can produce different answers depending on the operator.

For example:

- `/` returns the decimal result.
- `//` returns only the whole-number part.
- `%` returns the remainder.
