## 🌟 Biggest Lesson

Programming is not about memorizing code—it's about understanding how the computer thinks.

One concept that really clicked for me was that variables are like labeled boxes that store values, and that `print()` only displays the value stored in a variable at the moment it is called.

I also learned that `input()` always returns a string, even when the user enters numbers.

---

## 🧩 Biggest Challenge

One of my biggest challenges was understanding why updating a variable didn't change what had already been printed to the terminal.

For example:

```python
score = 90
print(score)

score = 100
print(score)
```

I expected only `100` to appear, but learned that `print()` displays the value at that specific point in the program. Updating a variable doesn't change output that has already been displayed.

I also learned to pay close attention to data types after discovering that:

- `"5" + "5"` becomes `"55"`
- `"5" * 3` becomes `"555"`
- `5 + 5` becomes `10`
