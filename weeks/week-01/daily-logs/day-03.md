## 🤯 Biggest Surprise
I discovered that `input()` always returns a string.
Even if I type 25, Python sees it as `"25"` until I convert it using `int()`.
I also learned that `"5" * 3` produces `"555"` instead of an error because Python repeats strings when multiplying them by an integer.
