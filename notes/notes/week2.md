## Day 2

### Comparison Operators

- `==` Equal to
- `!=` Not equal to
- `>` Greater than
- `<` Less than
- `>=` Greater than or equal to
- `<=` Less than or equal to

Comparison operators always return a Boolean value (`True` or `False`).

### Things I Learned

- `=` assigns a value.
- `==` compares values.
- `"10" == 10` returns `False` instead of an error.
- `"Python" != "python"` returns `True` because Python is case-sensitive.
- `True == 1` and `False == 0` both evaluate to `True` in Python.
